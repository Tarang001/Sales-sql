# 📊 Interactive Data Visualization Tool (Simulating BI Workflows)

## 🚀 Overview

This project is a web-based data visualization tool that allows users to upload CSV datasets and generate dynamic charts for exploratory data analysis. It simulates core Business Intelligence (BI) functionalities such as data transformation, visualization, and dashboard-like interaction.

---

## 🎯 Features

* 📂 Upload custom CSV datasets for analysis
* 📊 Generate dynamic charts and visual insights
* 🔄 Real-time data processing and transformation
* 🧪 Demo dataset support for quick exploration
* 🖥️ Interactive UI for flexible data exploration
* 📈 Simulates BI workflows and dashboard experience

---

## 🧠 Key Concepts Demonstrated

* Data Cleaning & Transformation
* Exploratory Data Analysis (EDA)
* Data Visualization Techniques
* Simulating BI tools (like Power BI)
* User-driven data interaction
